# Global Commerce OS

Live Demo: https://bijualbert.com  

## Overview
Global Commerce OS is a Shopify-inspired, headless commerce platform supporting
multi-region inventory, pricing, automation, and analytics.

## Tech Stack
- React / TypeScript
- Node.js
- PostgreSQL
- Drizzle ORM
- Lovable (hosting)
- Hydrogen-style headless storefront

## Running Locally
npm install
npm run dev
